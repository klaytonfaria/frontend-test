# Footer

Follow settings and markup example of how to use `footer` component:

### Style variations:

| class   | description               |
|---------|:------------------------- |
| fixed   | Turn footer display fixed |

### Base markup:
```html
<div class="footer-container">
  <footer class="page-footer">
      <div class="content-wrapper">Footer content</div>
  </footer>
</div>
```
