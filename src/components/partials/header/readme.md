# Header

Follow settings and markup example of how to use `header` component:

### Style variations:

| class   | description               |
|---------|:------------------------- |
| dark    | dark color                |
| fixed   | Turn header display fixed |

### Base markup:
```html
<div class="header-container">
  <header class="page-header">
      <div class="content-wrapper">Header content</div>
  </header>
</div>
```
